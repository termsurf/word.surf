Here are some ways in which to express complex "nouns" coming from
English.

| english   | breakdown      | read        |
| :-------- | :------------- | :---------- |
| evolution | evolve process | valv kix    |
| exception | except result  | Coz tak     |
| execution | execute action | sekyut kiq  |
| executive | execute agent  | sekyut zek  |
| existence | exist state    | wav dom     |
| expansion | expand action  | hariv kiq   |
| expertise | expert state   | zandjam dom |
| extension | extend result  | dem tak     |
| formation | form result    | form tak    |
| historian | history agent  | lixik zek   |
| indicator | indicate tool  | simun tul   |
| infection | infect result  | ganran tak  |
