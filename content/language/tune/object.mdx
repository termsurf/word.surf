## Formal Names

Other formal names include place names and species names, amongst other
things.

> The Big Owl-Eyed Night Walker.

So it would be broken down as:

```
Big   Owl    -Eyed     Night -Walk  -er.
brat  yanxuf leg nayan nit   wak  zeka
```

So it can get arbitrarily complex that way.

A "gorilla" is a "large monkey" (`brat maqk`). To know we aren't just
talking about generic large monkeys, we can prefix the noun phrase with
`kaq` signifying "type".

```
kaq brat maqka
```

```
The beautiful gorilla        is   a   large monkey.
dan fal byut  kaq brat maqka vuti nic brat  maqka.
```

## Object Terms

Here are some example more complex multi-word terms.

Ideally formal noun-phrases only use max 3 words, but occasionally you
may need to use 4 or more.

Some things can't have a memorable name, like the name of specific
classes of rock, or of molecules or genes. You could theoretically have
a "simple" name, but it would be a long name with many simple words. So
instead of that, we can create short words. But you are unlikely to
memorize more than a few dozen words in each category, so there is a
tradeoff. But for formal sciences where they need an identifier, they
can use a longer formal word. Ideally, however, we have a short
descriptive word, like the following:
