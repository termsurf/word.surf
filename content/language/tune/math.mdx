## Mathematical Expressions

Here is an example of how you would start writing mathematical
expressions in Tone. They are simply trees of statements.

| expression                                            |
| :---------------------------------------------------- |
| $e^x$                                                 |
| gal(yular, p)                                         |
| <T text="gal(yular, p)" />                            |
| $a^2 + b^2 = c^2$                                     |
| zig lax(gal(p, D), gal(k, D)), gal(b, D)              |
| <T text="zig lax(gal(p, D), gal(k, D)), gal(b, D)" /> |
